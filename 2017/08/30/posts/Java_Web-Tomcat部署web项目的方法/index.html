Tomcat部署web项目的方法

发表于 2017-08-30 | 更新于 2018-12-13 | 分类于 Java Web

内部应用

直接把war包复制粘贴到Tomcat 8.5/webapps目录下, 然后执行Tomcat 8.5/bin/startup.bat即可。

外部应用

  • docBase: 项目文件夹实际的位置, 子目录为WEB-INF、META-INF的一个目录。
  • path: 虚拟路径, 浏览器访问本项目的路径, 上面例子为http://本机地址:端口/hello
