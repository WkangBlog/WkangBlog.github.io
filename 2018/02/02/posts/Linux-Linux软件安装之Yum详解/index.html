Linux软件安装之Yum详解

发表于 2018-02-02 | 更新于 2018-12-13 | 分类于 Linux

前言

Yum(Yellow dog Updater, Modified)是一个基于RPM包管理的字符前端软件包管理器。
能够从指定的服务器自动下载RPM包并且安装，可以自动处理依赖性关系，并且一次安装所有依赖的软件包，无须繁琐地一次次下载、安装。
不支持RPM查询和校验。
