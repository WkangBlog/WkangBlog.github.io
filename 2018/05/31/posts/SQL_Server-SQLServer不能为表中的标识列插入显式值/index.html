SQLServer不能为表中的标识列插入显式值

发表于 2018-05-31 | 更新于 2018-12-13 | 分类于 SQL Server

前言

今天遇到个问题，要把线上生产环境的某张数据表的数据导出到本地的测试环境，但是导出后的SQL文件执行时出现了错误。

解决方案

设置identity_insert为ON，执行完毕再设置为OFF即可。
设置语法为SET IDENTITY_INSERT [ database.[ owner.] ] { table } { ON | OFF }

参考资料

  • 当 IDENTITY_INSERT 设置为 OFF 时，不能为表中的标识列插入显式值
