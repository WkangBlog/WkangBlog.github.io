使用SLF4J在多线程下输出到不同的日志文件

发表于 2018-10-16 | 更新于 2018-12-13 | 分类于 Java Web

前言

最近做了个数据传输的模块, 用到了线程池多线程, 要求将市平台的文章导入到公司的项目中进行管理, 每个栏目都有对应的文章, 但是打印日志时出现了日志混乱的问题。

思路

每个线程输出到不同的日志文件下, 这个功能应该是可以做到的, 只要在初始化Logger时指定对应的文件名就可以了。
